# Fleet Anomaly Detection System

## Objective

Build an AI-powered anomaly detection platform
for fleet telemetry data.

## Use Cases

- Overspeed Detection

- Engine Temperature Anomalies

- Fuel Consumption Anomalies

- Route Deviations

- Excessive Idling

- Maintenance Risk Detection

## Technologies

- Python

- Scikit-learn

- XGBoost

- PostgreSQL

- FastAPI

- MLflow

- Streamlit
